# Sistema de Gestão - Maria Gabriella Bento 💅

Sistema web desenvolvido para gestão financeira e histórico de atendimentos de estética.

### 🚀 Funcionalidades
* **Lançamentos:** Registo de serviços com cálculo automático de comissão (30% dias úteis / 20% domingos).
* **Histórico de Clientes:** Filtro por cliente com opção de editar nomes em massa.
* **Relatórios:** Geração de PDF diário e mensal com data por extenso e rodapé personalizado.
* **Dashboard:** Gráficos interativos de faturação e serviços mais realizados.

### 🛠️ Tecnologias
* HTML5 / CSS3 (Design Responsivo)
* JavaScript (ES6+)
* Firebase (Firestore & Auth)
* Chart.js (Gráficos)
* jsPDF (Relatórios)
